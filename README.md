## 📖 About

**Madhu Educational Academy (MEA)** is a public-interest education guidance platform based in **Kadapa, Andhra Pradesh, India**. We are not a coaching centre. We do not sell courses. We help students, parents, professionals, and educators **cut through confusion** and make clear, informed decisions about learning and careers — **without the hype**.

> *"We help you decide — not tell you what to do."*

This repository contains the static website for the **Decision Compass** platform — a hype-free, India-grounded resource built on honesty, realism, and lifelong learning principles.

## 🏛️ Six Pillars of Educational Clarity

```
┌─────────────────────────────────────────────────────────────┐
│  01 · Academic & Career Decision Clarity   [Most Popular]   │
│  02 · Education System Literacy            [Essential]       │
│  03 · Lifelong Learning & Career Transitions [Life-Stage]   │
│  04 · Academic & Teaching Careers          [Coming Soon]     │
│  05 · Skills, Credentials & Certifications [High Demand]    │
│  06 · Learning How to Learn                [Self-Dev]        │
└─────────────────────────────────────────────────────────────┘
```

### 01 🎓 Academic & Career Decision Clarity
Choices after 8th / 10th / 12th · Degree vs diploma vs skill certificate · Drop year decisions · When NOT to pursue higher education · Navigating parent expectations vs student aptitude

### 02 🏫 Education System Literacy
Central vs State universities · UGC / AICTE / NAAC / NBA explained · NEP simplified · Online education validity · How to verify institution legitimacy

### 03 🔄 Lifelong Learning & Career Transitions
Career breaks and recovery · Learning at 30 / 40+ · Homemaker re-entry pathways · Balancing study with family · Slow learning and sustainable growth

### 04 👩‍🏫 Academic & Teaching Careers
Teaching as a long-term career · PhD realities · NET / SET purpose and limitations · Academic vs industry trade-offs · Managing burnout in academia

### 05 📜 Skills, Credentials & Certifications
Online certification credibility · Credential inflation effects · Employer perception of certificates · When certificates actually help — and when they don't

### 06 🧠 Learning How to Learn
Self-assessment of learning needs · Planning long-term learning paths · Using free resources effectively · Unlearning outdated knowledge · Becoming a lifelong self-learner

## 🚀 Deployment

### GitHub Pages (Recommended)

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://yourusername.github.io/repository-name`

### Local Preview

```bash
# No server needed — just open in browser
open index.html

# Or use Python's built-in server
python -m http.server 8000
# Then visit http://localhost:8000
```

## 🙏 Our Philosophy

We built this because honest education guidance should not be a privilege. A first-generation college student in rural Andhra Pradesh deserves the same quality of thinking-support as someone with a well-connected family in a metro city. **That is the entire mission.**

The name *Decision Compass* — not *Decision Map* — is intentional. A compass shows direction. It does not choose the destination for you.
